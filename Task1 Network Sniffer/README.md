# Network Sniffer

## Project Overview
The Network Packet Sniffer is a Python-based cybersecurity tool that captures and analyzes network packets in real time. It helps users monitor network traffic, inspect packet details, and understand communication between devices on a network.

## Features
- Capture live network packets
- Display source and destination IP addresses
- Show network protocols used
- Analyze packet information in real time
- Lightweight and easy to use
- Educational tool for learning network security concepts

## Technologies Used
- Python
- Scapy Library
- Npcap (Windows)
